## What's this?

A simple app to help you reduce your tinnitus using ACRN (= [Acoustic Coordinated Reset Neuromodulation](https://www.sciencedirect.com/science/article/pii/S1053811913002553)) treatment.

This app is heavily inspired by [headphonejames/acrn](https://github.com/headphonejames/acrn).
I would like to extend my heartfelt appreciation to [headphonejames/acrn](https://github.com/headphonejames/acrn) for his outstanding work which has been a great source of inspiration for this app.

## About development

You can find the source code of this app on [GitHub](https://github.com/r4ai/tinnitus-reducer-acrn).

### Technology stack

- Framework: [Tauri](https://tauri.studio/)
- Frontend:
  - Language: TypeScript
  - Framework: Svelte
  - Audio: [tone.js](https://tonejs.github.io/)
  - UI: TailwindCSS, DaisyUI, svelte-range-slider-pips
  - Icons: Fluent Emoji, Heroicons
- Backend:
  - Language: Rust
